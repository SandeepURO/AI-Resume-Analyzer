#  AI Resume Analyzer

An AI-powered Resume Analyzer built using Python, Flask, HTML, CSS, and JavaScript. The application analyzes resumes, calculates ATS scores, extracts skills, generates suggestions, and matches resumes against job descriptions.

##  Features

* Upload PDF resumes
* Extract text from resumes automatically
* Detect technical skills
* Detect education details
* Detect projects and experience
* Generate ATS compatibility score
* Provide resume improvement suggestions
* Match resumes against job descriptions
* Identify missing skills for target jobs
* Professional and responsive user interface

##  How It Works

1. User uploads a resume.
2. Resume is sent to the Flask backend.
3. PDF text is extracted.
4. Skills, education, projects, and experience are detected.
5. ATS score is calculated.
6. Resume suggestions are generated.
7. Job description matching is performed.
8. Missing skills are identified.
9. Results are displayed on the dashboard.
