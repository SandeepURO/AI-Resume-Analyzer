# AI Resume Analyzer

An AI-powered Resume Analyzer built using Python, Flask, HTML, CSS, and JavaScript.

## Features

* Upload PDF resumes
* Extract resume text automatically
* Detect technical skills
* Detect education details
* Detect projects and experience
* Generate ATS compatibility score
* Provide resume improvement suggestions
* Match resumes against job descriptions
* Identify missing skills for target jobs

## Technologies Used

* Python
* Flask
* HTML
* CSS
* JavaScript
* PyPDF2
* Git & GitHub

## How to Run

1. Clone the repository
2. Create and activate a virtual environment
3. Install dependencies
4. Run:

```bash
python app.py
```

5. Open:

```text
http://127.0.0.1:5000/frontend
```
